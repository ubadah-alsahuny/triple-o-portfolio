# Company Frontend

This is the frontend part of the My Project application, built with React js.

---

## Prerequisites

Node .js (version 20.17.0)

npm (version 9.7.2)

---

## Installation

1. Clone the repository: if you haven't done it already in the backend Readme , feel free to copy & paste the code below

```bash
git clone https://gitlab.com/Hedara444/protofolio.git

```

2. Navigate to the frontend directory

```bash
   cd protofolio-frontend
```

3. install the dependencies

```bash
npm install
npm install bootstrap
npm install tailwindcss @tailwindcss/vite
npm install react-icons
npm install lucide-react
npm install --save-dev @types/jspdf
npm install fabric
npm install fabric jspdf html2canvas
```

npm install react-konva konva jspdf use-image
npm uninstall fabric

4. run the application

```bash
npm run dev
```

The application will be available at http://localhost:5173 .
